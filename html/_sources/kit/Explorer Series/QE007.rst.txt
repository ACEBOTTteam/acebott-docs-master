QE007 IOT Smart Home Starter Kit for ESP32
====================

一、Product Introduction
------

Lumi's Home is an intelligent home system that combines features such as Wi-Fi control, voice control, and app control. 

It incorporates interactive controls such as RFID door access, automatic window closure when it rains, 
and control over LED lights, RGB and more.
