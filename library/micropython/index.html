MicroPython类库

以下库中提供了特定于MicroPython-ESP32实现的功能。

  • btree – 简单的 BTree 数据库
  • framebuf — 帧缓冲操作
  • machine — 与硬件相关的功能
  • network — 网络配置
  • ubluetooth — 低功耗蓝牙
  • ucryptolib – 加密密码
  • uctypes – 以结构化方式访问二进制数据
  • micropython – 访问和控制MicroPython内部
  • neopixel — WS2812 灯带
  • random — 生成随机数
  • ntptime — 时间同步
  • dht — dht模块
  • _thread — 线程
  • esp — ESP32相关的功能
  • esp32 — 特定于ESP32的功能
  • onewire — 单总线
